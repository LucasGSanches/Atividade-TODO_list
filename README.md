# Atividade-TODO_list
Atividade universitária da disciplina de desenvolvimento web sobre html, css e javascript.

**Descrição breve do projeto**

O projeto utiliza de html, css e javascript, bem como algumas boas práticas como css reset e html semântico afim de 
fazer um site sobre "todo-list"(lista de afazeres).

**Instruções de como executar a aplicação**

Para utilizar o site, o usuário deve selecionar a entrada de texto para colocar a descrição da atividade,
e depois apertar no botão "add" para adicionar a tarefa na lista. Para continuar adicionando tarefas basta
repetir os passos anteriores, e caso deseja remover alguma tarefa, basta clicar no "X" na tarefa desejada.
